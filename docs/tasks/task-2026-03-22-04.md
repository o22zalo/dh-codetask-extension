### Thực hiện tiếp tục:

- [ ]: Chỉnh lỗi: Tìm theo theo nội dung không có kết quả nào hết mặc dù đã cấu hình ("RipgrepPath": "C:\\ripgrep\\rg.exe")
- [ ]: Bổ sung thêm log lệnh tìm kiếm khi thực hiện tìm kiếm trong output window, log rõ ràng lệnh chạy, thời gian thực thi, và kết quả output của Ripgrep (để xem log và dễ dàng điều chỉnh thay đổi khi cần)
- [ ]: Chức năng năng làm tươi trong task history panel chưa được,
  - phải thoát mở lại mới load các danh sách file mới, chức năng checksum chưa đúng yêu cầu,
  - hiện json có field checksum thì bị ghi nhận là thay đổi, mặc dù chưa hề chỉnh sửa file này, chỉ có extension thay đổi thôi. (như vậy là chưa đúng yêu cầu).
  - Phần thể hiện thơi gian chưa lấy các thông tin từ file lên
- [ ]: Phát triển thêm tính năng TODO List:
  - Dựa vào TodoTemplates trong settings để thêm chức năng chọn từ danh sách, đồng thời chỉnh button `add` ở trạng thái `enabled`.

  ```
  "TodoTemplates": [
  "Xác định có lỗi như yêu cầu",
  "Tìm nguyên nhân lỗi",
  "Chỉnh lỗi",
  "Kiểm thử lại chức năng"
  ]
  ```

  - Điều chỉnh các button điều kiển todo list không có tác dụng, và đúng trạng thái, ví dụ start rồi, thì không start nữa, disable đi, khi start rồi mới có tạm ngưng, đã xong cũng theo luồng qui tắc này.
